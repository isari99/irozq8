� طلب تعديل لعبة XO (إعداد نظام اللعب + الواجهة):

أبغى تنفيذ التعديلات التالية بدقة بدون أخطاء:



🖼️ 1️⃣ واجهة البداية (اللوغو والصفحة الرئيسية):

    •    إزالة النص الحالي: 
    ◦    “لعبة XO الكلاسيكية الخالدة بين فريقين” بالكامل.
    •
    •    إزالة أي نصوص مرفقة مع اللوغو الحالي.
    •    استبدال اللوغو بصورة جديدة سأقوم بإرسالها.
    •    الصورة الجديدة تكون: 
    ◦    داخل مربع كامل (Square).
    ◦    ومضبوطة بشكل متناسق وممتلئة 100% داخل الإطار بدون قص أو فراغات.
    •
    •    زر “ابدأ اللعب” (لعب الآن): 
    ◦    يكون تحت الصورة مباشرة (ليس داخل الصورة).
    ◦    نفس أسلوب زر لعبة الشخصنة.
    ◦    زر واضح وجذاب ويؤدي إلى بدء اللعبة.
    •




👥 2️⃣ نظام الدخول (Join):

    •    عند كتابة join في الشات: 
    ◦    يتم إدخال اللاعبين تلقائيًا.
    •
    •    يتم قبول أول شخصين فقط: 
    ◦    أول لاعب = X
    ◦    ثاني لاعب = O
    •
    •    بعد اكتمال لاعبين: 
    ◦    يتم إغلاق الانضمام (لا يقبل لاعبين إضافيين).
    •
    •    عرض صور اللاعبين (الأفاتار) في واجهة الانتظار.




▶️ 3️⃣ بدء اللعبة:

    •    عند الضغط على زر ابدأ اللعب: 
    ◦    يتم تحويل اللاعبين إلى صفحة XO.
    ◦    تبدأ المباراة بين X و O فقط.
    •




🔢 4️⃣ واجهة XO (المربعات):

    •    عرض شبكة XO بشكل واضح.
    •    كل مربع يحتوي على رقم: 
    ◦    1، 2، 3، 4، 5، 6 … (حسب عدد المربعات).
    •
    •    الأرقام تكون واضحة وبسيطة بدون تعقيد.




🎮 5️⃣ آلية اللعب:

    •    اللاعب يكتب الرقم في الشات لاختيار المربع.
    •    يتم وضع العلامة داخل المربع: 
    ◦    X للاعب الأول
    ◦    O للاعب الثاني
    •
    •    منع اختيار مربع تم استخدامه مسبقاً.




🔄 6️⃣ نظام الأدوار:

    •    البداية تكون للاعب X.
    •    يتم التبديل تلقائياً بعد كل دور: 
    ◦    X ثم O ثم X وهكذا.
    •
    •    عرض الدور الحالي بشكل واضح في الواجهة.
    •    منع اللاعب من اللعب خارج دوره.




🖼️ 7️⃣ عرض الصور:

    •    عند دخول اللاعبين: 
    ◦    تظهر صورهم (الأفاتار).
    •
    •    أثناء اللعب: 
    ◦    يتم إبراز اللاعب الحالي (الدور X أو O)..
🏆 8️⃣ الفوز:

    •    عند فوز أحد اللاعبين: 
    ◦    يتم إعلان الفائز.
    ◦    عرض صورة اللاعب الفائز.
    ◦    إظهار رسالة فوز واضحة.
    •




🔁 9️⃣ إعادة المباراة (Rematch):

    •    عند انتهاء اللعبة (فوز أو تعادل): 
    ◦    لا يتم إخراج اللاعبين من الجيم.
    ◦    يتم عرض خيار/زر إعادة المباراة.
    •
    •    عند اختيار إعادة المباراة: 
    ◦    يتم تفريغ اللوحة وإعادة اللعبة من البداية.
    ◦    إعادة تعيين الأدوار بحيث يبدأ اللاعب X.
    ◦    يتم الاحتفاظ بنفس اللاعبين (X و O) بدون الحاجة لكتابة join مرة أخرى.
    •
    •    يمكن تكرار إعادة المباراة أكثر من مرة بنفس اللاعبين.




🎯 الهدف:

    •    بناء لعبة XO كاملة تعمل عبر الشات.
    •    تحسين واجهة البداية باستخدام صورة مخصصة وزر لعب واضح.
    •    تنظيم نظام الدخول للاعبين فقط (2 لاعبين).
    •    عرض الأدوار والمربعات بشكل واضح.
    •    دعم إعادة اللعب بدون إعادة انضمام.
    •    جعل التجربة احترافية وسلسة
