
[تعديل نهائي على لعبة الكراسي + نظام الصور + الجودة]

1) تحسين جودة اللوبي (Lobby):
المشكلة:
• الصفحة ضعيفة (النصوص مو واضحة + اللاعبين مو واضحين)

المطلوب:
• تكبير الخط وجعله واضح
• تحسين تصميم الأزرار
• تكبير صور اللاعبين
• ترتيب العناصر بشكل نظيف
• جودة عالية مشابهة لألعاب XO (واجهة واضحة وبسيطة)

-----------------------------------

2) تعديل نظام الصور (مهم جداً - لكل الألعاب):
المشكلة:
• النظام يحط صورة Twitch افتراضية

المطلوب:
• عند كتابة join:
يتم سحب صورة المستخدم من Twitch API
استخدام profile_image_url
• عرض الصورة الحقيقية لكل لاعب
• تطبيق هذا على جميع الألعاب (مو بس الكراسي)
• عدم استخدام صورة افتراضية إلا عند فشل الطلب

-----------------------------------

3) تحسين شاشة اللعبة (Game UI):
المطلوب:
• تحسين الجودة العامة (وضوح + نظافة)
• تكبير العناصر بالكامل
• جعل التصميم مشابه لجودة لعبة XO
• عرض صورة "music" داخل العجلة بشكل واضح

-----------------------------------

4) تعديل العجلة (Wheel):
المشكلة:
• صغيرة + سريعة + مو واضحة

المطلوب:
• تكبير العجلة بشكل كبير وواضح
• تقليل سرعة الدوران (Smooth)
• عرض صور اللاعبين داخلها بوضوح
• إزالة أي عناصر تشوش على الرؤية

-----------------------------------

5) إزالة التحكم اليدوي:
المشكلة:
• يوجد زر "إيقاف العجلة"

المطلوب:
• حذف زر إيقاف العجلة نهائياً
• التحكم يكون تلقائي فقط

-----------------------------------

6) نظام الوقت والموسيقى (Game Logic):
المطلوب:
• عند بدء الجولة:
تشغيل الأغنية
بدء دوران العجلة

  • تحديد وقت عشوائي لكل جولة مثل:
      
10 ثواني
15 ثانية
20 ثانية
22 ثانية

  • عند انتهاء الوقت:
      
تتوقف العجلة تلقائياً
تتوقف الأغنية مباشرة
تظهر الكراسي

-----------------------------------

7) الكراسي (Chairs):
المطلوب:
• عرض الكراسي بتصميم جميل
• وضع أرقام واضحة على كل كرسي
• توزيعها بشكل مرتب وواضح

عند اختيار اللاعب:
• اللاعب يتحرك بشكل تدريجي (Animation)
• يمشي للكرسي مو ينتقل فجأة
-----------------------------------

8) تحسين التجربة العامة:
جودة عالية في كل الصفحات (Lobby / Game / Winner)
وضوح كامل في النصوص والصور
حركة سلسة (Animations)
بدون لاج أو تقطيع

-----------------------------------

[مهم جداً]
الصور = من Twitch لكل اللاعبين (في جميع الألعاب)
العجلة = كبيرة + واضحة + بطيئة نسبياً
الموسيقى = مرتبطة بالوقت (توقف مع العجلة)
لا يوجد زر إيقاف يدوي
الكراسي = واضحة + مرقمة + بتصميم جميل
