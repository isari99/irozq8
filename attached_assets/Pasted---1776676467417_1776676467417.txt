
│ تعديل لعبة الكراسي الموسيقية                                              │
├──────────────────────────────────────────────────────────────────────────────┤
│ 1) تحسين وضوح النصوص داخل اللعبة                                           │
│ - جعل كل النصوص (join / اللاعبين / الرسائل) واضحة وحادة مثل لعبة XO       │
│ - رفع دقة الخطوط + زيادة التباين                                           │
│                                                                              │
│ 2) صفحة بدء اللعبة                                                          │
│ - إصلاح عرض النص:                                                           │
│   "join in chat to join the musical chairs game"                           │
│   "players joined: 0"                                                       │
│   "not enough players yet"                                                  │
│ - تكون واضحة 100% بدون تشويش أو ضعف قراءة                                   │
│                                                                              │
│ 3) العجلة (Wheel)                                                           │
│ - رفع حدة الإضاءة والألوان بالكامل                                         │
│ - جعل الألوان قوية وواضحة مثل لعبة XO                                      │
│ - تحسين الإضاءة داخل العجلة وعدم وجود أي خفوت                               │
│                                                                              │
│ 4) شكل الكراسي داخل اللعبة                                                 │
│ - إضافة تصميم كرسي واضح ومرتب لكل خانة                                     │
│ - وضع رقم فوق كل كرسي بشكل
واضح                                            │
│                                                                              │
│ 5) حركة اللاعبين                                                           │
│ - عند اللعب اللاعب يتحرك تلقائيًا نحو الكرسي بشكل ناعم (smooth movement)   │
│                                                                              │
│ 6) الصوت                                                                    │
│ - إضافة تحكم بالصوت داخل اللعبة                                             │
│ - زر رفع وخفض صوت الموسيقى أثناء اللعب                                     │
│                                                                              │
│ 7) شرط مهم                                                                  │
│ - ممنوع أي ضعف جودة أو تشويش أو ألوان باهتة                                │
│ - لازم نفس جودة لعبة XO بالضبط (Sharp + Clear + High Contrast)             │
└────────────────────────────────────────────────────────────────────────────
